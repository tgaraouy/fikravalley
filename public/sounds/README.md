# Notification Sounds

Place your notification sound file here as `notification.mp3`.

## Recommended Audio File

- **Format**: MP3 (most compatible)
- **Duration**: 0.5-2 seconds (short notification)
- **Volume**: Normalized to avoid sudden loud sounds
- **Sample Rate**: 44.1kHz or 48kHz

## Free Sound Resources

You can find free notification sounds at:
- [Freesound.org](https://freesound.org) - Search for "notification" or "beep"
- [Zapsplat](https://www.zapsplat.com) - Free sound effects
- [Mixkit](https://mixkit.co/free-sound-effects/notification/) - Free notification sounds

## Example

If you don't have a file, the dashboard will automatically fall back to a Web Audio API beep sound.
