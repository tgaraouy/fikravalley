## QUICK TIPS

1. **Image Quality:** Generate at 2x resolution (3840x2160) then scale down for crispness
2. **Consistency:** Use same style prompts for similar slides
3. **Colors:** Always include Moroccan flag colors (#C1272D, #006233)
4. **Text Overlay:** Ensure text is readable over images (use semi-transparent backgrounds)
5. **Backup:** Save PowerPoint file frequently
6. **Export:** Create PDF version for sharing
