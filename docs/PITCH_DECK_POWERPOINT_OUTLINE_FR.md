## SLIDE 2: LE PROBLÈME

### Layout
Infographie split-screen. Gauche : La lutte. Droite : Les barrières et données.

### Description Visuelle & Prompt NANO BANANA PRO
**Côté Gauche :** Image de femme marocaine frustrée avec ordinateur portable/cadre traditionnel

**Côté Droite :** Icônes pour langues, coûts, chaîne de mentors cassée, et visualisation de données de taux d'échec de 90%

```
Infographie montrant trois barrières pour les entrepreneurs marocains, design split-screen, côté gauche montre femme marocaine frustrée avec ordinateur portable dans cadre traditionnel, côté droit montre icônes pour langues (Darija/Tamazight), barrières de coût (5K-10K$), chaîne de mentors cassée, et visualisation de données de taux d'échec de 90%, rouges atténués pour problèmes, vert vif pour opportunité, design plat moderne, ratio 16:9
```

### Contenu

**LE DÉFI : LES GRANDES IDÉES MEURENT NON TESTÉES**

Les entrepreneurs marocains font face à 3 barrières critiques :

**Gap de Validation**

- 90% des idées échouent car elles ne sont pas validées avant de construire.
- La recherche de marché est coûteuse (5K-10K$) et difficile d'accès.

**Barrières Linguistiques**

- Les plateformes traditionnelles ne supportent que l'anglais/français.
- 70% des Marocains préfèrent la voix à la saisie (Darija/Tamazight).

**Accès aux Mentors**

- Mentorat local limité ; expertise de la diaspora reste inexploitée.
- Aucun système de matching structuré.

**Contexte de Marché :**

- Malgré la croissance de l'écosystème (94,96M$ levés en 2024, triplé depuis 2023), les tours individuels restent petits.
- Le Maroc est en retard derrière le Nigeria/Kenya en financement agrégé.
- 74% de pénétration internet crée une opportunité massive inexploitée.
