## DESIGN NOTES

### Color Scheme:
- **Primary:** Moroccan flag colors (Red #C1272D, Green #006233, Gold/Amber)
- **Background:** Light gradients (red-50, amber-50, green-50)
- **Text:** Dark slate for readability

### Typography:
- **Headlines:** Bold, large (32-48pt)
- **Body:** Medium (16-18pt)
- **Supporting:** Small (12-14pt)

### Visual Style:
- Clean, modern, professional
- Moroccan cultural elements (subtle)
- Icons for features
- Charts/graphs for data
- Screenshots for product

### Slide Count:
- **Full Deck:** 15 slides
- **Short Version:** 10 slides (remove tech, team, vision details)
- **Elevator Pitch:** 5 slides (Problem, Solution, Market, Traction, Ask)

## PRESENTATION TIPS

1. **Start Strong:** Problem slide should resonate emotionally
2. **Show, Don't Tell:** Use screenshots, demos, user stories
3. **Keep It Simple:** One idea per slide
4. **Practice Timing:** 15 slides = 15-20 minutes
5. **End with Ask:** Clear, specific, actionable
